# edoFundStats
Python program retrieving fund-statistics, store to local database and generate graphical report


----------------------
Installing
----------------------
```
# sudo apt-get py-reportlab
# sudo apt-get py-httplib2
# git clone https://github.com/engdan77/edoFundStats.git

```
